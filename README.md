# FindMySpot

Vehicle Parking App - V2

It is a multi-user app (one requires an administrator and other users) that manages different parking lots, parking spots and parked vehicles. Assume that this parking app is for 4-wheeler parking.

## Frameworks to be used
These are the mandatory frameworks on which the project has been built on.


Flask for API
VueJS for UI
VueJS Advanced with CLI 
Jinja2 templates if using CDN only for entry point 
Bootstrap for HTML generation and styling 
SQLite for database 
Redis for caching
Redis and Celery for batch jobs
